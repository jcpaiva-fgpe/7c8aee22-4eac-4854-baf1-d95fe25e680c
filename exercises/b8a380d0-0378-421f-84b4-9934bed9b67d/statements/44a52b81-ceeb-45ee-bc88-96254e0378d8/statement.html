Climate Data Preprocessing Challenge

The GCIA has provided a raw dataset "climate_change.csv" with multiple climate indicators. Your task is to prepare it for modeling.

  1. Load the dataset with its path into a DataFrame.
  2. Handle missing data:
    • Fill missing numeric values (Avg Temperature (°C), CO2 Emissions (Tons/Capita), Sea Level Rise (mm), Rainfall (mm), Population, Renewable Energy (%), Extreme Weather Events, Forest Area (%)) with the median of their respective column.
    • Drop any rows with missing Country or Year.
  3. Feature engineering:
    • Create CO2_per_Person = CO2 Emissions (Tons/Capita) × Population.
    • Create Sustainability_Index = (Renewable Energy (%) + Forest Area (%)) / 2.
  4. Deduplicate by mean (multiple entries per year):
    • Some countries have multiple rows for the same Year. Group the data by Country and Year and take the mean for all numeric columns.
    • This ensures that for each Country and Year combination there is only one entry, which represents the mean values.
    • After aggregation, reset the index so Country and Year are regular columns again.
    • Finally, order the dataset by Country and then by Year.
  5. Return the DataFrame.
