Climate Trends Across Countries

Using the prepared dataset, create a visualization of emissions for selected countries.

  1. Read the dataset into a DataFrame.
  2. Filter the data to include only the countries USA, China, and India.
  3. Plot a line chart of CO2 Emissions (Tons/Capita) over time:
    • X-axis: Year
    • Y-axis: CO2 Emissions (Tons/Capita)
    • title: CO2 Emissions per Capita Over Time
    • Y-axis: CO2 Emissions (Tons/Capita)
    • Each country should be drawn as its own line with markers 'o' and plot the legend.
