Flagging High Emitters

The GCIA wants to identify which countries are above the global average in 2040.

  1. Load the data from the path to the "emissions.csv" into a DataFrame named df.
  2. Calculate the mean of the 2040 column.
  3. Create a new column 2040_flag using .apply() and a lambda function:
    • "High" if the value is above the mean
    • "Low" otherwise
  4. Return the updated DataFrame showing the new column.
