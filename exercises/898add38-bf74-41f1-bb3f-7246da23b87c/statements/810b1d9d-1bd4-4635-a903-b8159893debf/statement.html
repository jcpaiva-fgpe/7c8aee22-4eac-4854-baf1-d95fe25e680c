Trade Sweep

Transaction volumes are flooding in from multiple exchanges. You must assemble them into a clear report.

  1. You are given an array sweeps such as [1250, 980, 1430, 1110].
  2. Write a loop that prints all results in one line.
  3. The output must look like:
    Exchange 1: 1250 BTC, Exchange 2: 980 BTC, Exchange 3: 1430 BTC, Exchange 4: 1110 BTC
  4. Make sure there is no extra comma at the end.
