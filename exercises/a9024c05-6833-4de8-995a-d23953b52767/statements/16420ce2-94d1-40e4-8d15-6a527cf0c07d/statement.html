Reshape Climate Data

Regional readings are coming in, but the agency needs them reorganized into clusters for analysis.

  1. Create an array of six warming values, for example: [0.9, 1.1, 1.3, 0.7, 1.0, 1.2].
  2. Reshape it into a 2x3 array, grouping the values into two clusters with three regions each.
  3. Return the reshaped array so the GCIA can analyze the clusters.
