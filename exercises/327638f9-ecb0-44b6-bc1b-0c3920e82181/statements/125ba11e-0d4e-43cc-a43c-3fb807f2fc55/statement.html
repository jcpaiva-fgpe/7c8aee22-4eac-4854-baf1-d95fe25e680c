Looping Through Sensors

You have multiple temperature sensors collecting data throughout the day. The readings are stored in a list, for example: [0.9, 1.2, 0.8, 1.5]. Write a loop that goes through each reading and prints them all in one single line like this: "Sensor 1: 0.9 째C, Sensor 2: 1.2 째C, Sensor 3: 0.8 째C, Sensor 4: 1.5 째C". Then print the sting.

Keep in mind to only have ',' inbetween the sensor readings.
