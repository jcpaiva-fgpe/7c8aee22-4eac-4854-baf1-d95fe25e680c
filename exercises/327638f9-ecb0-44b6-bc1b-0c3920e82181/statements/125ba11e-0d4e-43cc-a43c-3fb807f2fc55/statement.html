Sensor Sweep

Dozens of sensors report back from the field, and you must compile their data into a clean report.

  1. Given the variable sensor_array which is an array such as [0.9, 1.2, 0.8, 1.5]. Write a loop that prints all readings in one line.
  2. The output must look like:
    Sensor 1: 0.9 째C, Sensor 2: 1.2 째C, Sensor 3: 0.8 째C, Sensor 4: 1.5 째C
  3. Make sure there is no extra comma at the end.
