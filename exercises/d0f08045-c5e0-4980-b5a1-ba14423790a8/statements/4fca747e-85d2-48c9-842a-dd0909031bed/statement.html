Cleaning and Feature Engineering

The GCAA has delivered the raw "crypto_currency.csv" file. Before it can be used for modeling, it must be cleaned and structured.

  1. Load the dataset into a DataFrame named df.
  2. Clean numeric columns:
    • Remove % signs from the change columns (1h, 24h, 7d, 30d) and convert them to floats.
    • Remove $ and commas from Price, 24h Volume, and Market Cap and convert them to numeric values.
  3. Fill missing values:
    • For numeric columns: fill with the column median.
    • For categorical columns: fill with "Unknown".
  4. Feature engineering:
    • Create Daily_Volume_to_MarketCap = 24h Volume / Market Cap.
    • Create Return_30d = value from 30d column (already percentage) as float.
    • Create Liquidity_Index = 24h Volume / Circulating Supply.
