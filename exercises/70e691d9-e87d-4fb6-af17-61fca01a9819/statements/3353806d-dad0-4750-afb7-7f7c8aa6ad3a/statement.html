Combine Asset Returns

The GCAA has received daily return data for Bitcoin and Ethereum. You must calculate their combined impact across the same time period.

  1. You are given two arrays named bitcoin and ethereum (for example: bitcoin = [200, -300, 150], ethereum = [-100, 250, 50]).
  2. Add the arrays element-wise to calculate the total combined return for each day.
  3. Return the resulting array as your report.
